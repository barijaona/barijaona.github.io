Sushis en république Ranjalia

Antananarivo, Madagascar, entre Afrique et Asie, entre Dakar, Séoul et Tokyo.

dimanche 5 mai 2002
[ 22:13:49 ] 

Argh !! Pourquoi des gens comme moi continuent à vouloir faire dans la finesse dans ce monde de brutes ?

Entre la première HCC et la deuxième, pas moins de 4462 bureaux de vote diffèrent : entre les bureaux de vote qui figurent dans la deuxième et pas dans la première, les annulations partielles ou totales, et les chiffres qui diffèrent sans que l'on puisse comprendre pourquoi, il va falloir tenter de se faire une opinion...

[ 17:26:44 ] 

Contrôle de cohérence sur le nombre de suffrages exprimés :

  • les suffrages exprimés doivent être égaux aux nombres de votants moins les bulletins blancs et nuls
  • les suffrages exprimés sont la somme des suffrages obtenus par les candidats.

Ceci permet d'identifier encore quelques problèmes de décalages de données telles qu'affichées par les pages Web, notamment sur les Fivondronana d'Ambanja et d'Ambilobe (l'opérateur informatique manquait-il de sommeil ?)

[ 09:16:35 ] 

Pas de moyen simple de vérifier avec le nombre de bureaux de votes si une erreur matérielle ne s'est pas glissée dans les différents processus.

Voyons alors les bureaux de vote qui existeraient dans un fichier et pas dans l'autre.

Ceci permet de détecter une erreur de saisie dans les données de la HCC version 2 : Fivondronana Betroka, commune Iaborotra, bureau Fokontany Seheny. Le code bureau est 61310081 et non 31610081.

27 bureaux de vote existent dans la HCC2 et pas dans HCC1. En fait, ce nombre de bureaux de vote se ramène à 15, car les 12 autres ne montrent que des chiffres nuls (procès-verbaux de carence ou bureaux n'ayant pas ouvert ?).

Dans la plupart des cas, il semble s'agir de malheureux coups de "couper-coller" dans l'établissement de HCC1. Mais impossible d'en être sûr même en tentant de recouper les détails de l'un avec les récapitulatifs de l'autre...

Rajoutons dans notre base un champ : "Suspect"... Cela pourra servir plus tard à jouer l'avocat du diable.

[ 05:55:20 ] 

La codification des bureaux de vote de la HCC avait été très critiquée, car elle diffère de la codification publiée dans le Journal Officiel. Son principe : 3 chiffres pour le code Fivondronana, 2 chiffres pour le code Commune, et 3 chiffres pour identifier le bureau de vote dans la commune.

Je viens de finir de vérifier tous les bureaux de vote dont le code ne comporte pas 8 chiffres. Il y en a peu : quelques erreurs de saisie, surtout dans les données de la HCC version 1, et des problèmes de mise en page des pages Web (problème déjà évoqué sur les outils choisis par la HCC). En tout cas, aucune ambigüité. J'aligne la numérotation sur la HCC version 2.

Pas de doublons non plus dans la numérotation des bureaux.

Sauvegarde et petit dodo.
