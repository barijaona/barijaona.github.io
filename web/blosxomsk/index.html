Le kit de démarrage Blosxom est un assemblage de scripts Perl et de modèles de fichiers. Pour quelqu'un qui est intéressé pour créer un site généré dynamiquement ou statistiquement, c'est une base de départ intéressante. Téléchargement.

The Blosxom starter kit is a carefully crafted collection of Perl scripts, plugins and templates. If you want to have a site rendered either dynamically or statically, it is a good starting point. Download.
