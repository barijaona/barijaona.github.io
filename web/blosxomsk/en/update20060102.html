[ 23:59  ] New version of Blosxom Starter Kit

Late Christmas gift is available for download...

It is based on a recent version of blosxom.cgi (CVS revision 1.9) which has removed many bugs, and as a consequence required to change some plugins :

  • some hacks in flatarchives and categories plugins were removed because they are no more required
  • updated prevnextstory and pagetype plugins to handle the fixed $path_info information
  • updated the exclude plugin to the new method of handling the filter subroutines (much cleaner for static rendering !)

This version also includes the recent version of prefs plugin which is more friendly in case of errors in preferences files.

Plugins version numbers :

  • prevnextstory : 2006-01-01blosxom2
  • pagetype ; 2006-01-01blosxom2
  • exclude : 2006-01-02blosxom2
  • prefs : 2005-11-20

Reminder : the most recent release will always be available at http://blog.barijaona.com/download/blosxomsk.zip
